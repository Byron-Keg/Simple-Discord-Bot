# Simple Discord Bot  
A lightweight, beginner‑friendly Discord bot preset designed to help new developers get started quickly.  
This repository includes a clean project structure, environment variable setup, and a full step‑by‑step guide with images.

---

## 🚀 Features
- Modern Discord bot template using **discord.py**
- `.env` token loading for security
- Easy to follow setup steps with screenshots
- Perfect for beginners or quick prototyping

## ▶️ Installing Dependencies

Run this in your terminal:

```
pip install -r requirements.txt
```

---

## ▶️ Running the Bot

Launch the bot with:

```
python main.py
```

---

## 🔒 Security Notice
- **Never share your real bot token.**
- **Never commit your `.env` file.**
- If your token is ever exposed, reset it immediately in the Developer Portal.
